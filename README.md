Connects to your MySQL DB and runs the Assignment 6 transactions:

## Setup
```bash
python -m venv .venv
source .venv/bin/activate   # macOS/Linux
pip install -r requirements.txt
cp .env.example .env
```

## Run
```bash
flask --app app run --debug
```
Open http://127.0.0.1:5000
